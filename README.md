# vue-demo

> 基于: vue-2.5.2

> UI框架: element-ui - 1.4.7

> 路由管理: vue-router

> 状态管理: vuex

> http: axios

> 时间处理: moment

> 工具库: lodash

> 提示: noty

> 代码风格: eslint(airbnb)


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
